# Campus Navigator

A modern web application to help students navigate campus life with ease.

## About

Campus Navigator is a React + TypeScript application built to provide students with a seamless experience for exploring and navigating campus resources.

## Tech Stack

- **Vite** - Fast build tool and dev server
- **React 18** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Reusable component library
- **React Router** - Client-side routing
- **React Query** - Server state management

## Getting Started

Make sure you have **Node.js** and **npm** installed, then:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate into the project
cd campus-navigator

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:8080`.

## Available Scripts

| Command | Description |
|---|---|

| `npm run build` | Production build |
